<h1>
   Projeto: Segurança Digital no WhatsApp
</h1>

<p>
    Este repositório contém o código-fonte de um infográfico interativo (Página Web Funcional) desenvolvido como parte de um Projeto de Extensão Universitária. O objetivo principal é a capacitação de usuários, especialmente o público idoso, para a prevenção contra golpes de engenharia social no WhatsApp, com foco no golpe do "Falso Parente" via PIX.
</p>

<h2>Sobre o Projeto</h2>
<p>
    Esta ferramenta foi criada a partir de uma pesquisa que identificou a vulnerabilidade de usuários idosos frente a mensagens fraudulentas que solicitam transferências financeiras (PIX) de forma urgente. 
</p>
<p>
    A página web não é apenas um infográfico, mas uma solução de aprendizado interativo que guia o usuário através de 5 passos essenciais de verificação:
</p>

<ol>
    <li><strong>Alerta:</strong> Reconhecer a abordagem inicial.</li>
    <li><strong>Pressão:</strong> Identificar a tática de urgência do golpista.</li>
    <li><strong>Pausa:</strong> A importância de parar e respirar.</li>
    <li><strong>Verificação:</strong> O passo crucial de ligar para o número antigo.</li>
    <li><strong>Decisão:</strong> A ação correta de bloquear e não transferir.</li>
</ol>

<h3>Recursos Adicionais</h3>
<ul>
    <li><strong>Analisador de Risco (IA):</strong> Uma ferramenta que utiliza a API do Gemini para analisar mensagens suspeitas coladas pelo usuário.</li>
    <li><strong>Dados e Relatos:</strong> Contextualização do problema com estatísticas e relatos anônimos.</li>
    <li><strong>Recursos de Mídia:</strong> Inclui um podcast e um infográfico estático sobre o tema.</li>
</ul>

<h2>Tecnologias Utilizadas</h2>
<p>Este projeto foi construído intencionalmente como um arquivo HTML único para máxima portabilidade e facilidade de hospedagem (via GitHub Pages):</p>
<ul>
    <li><strong>HTML5:</strong> Estrutura semântica.</li>
    <li><strong>Tailwind CSS:</strong> Framework CSS (carregado via CDN) para design responsivo.</li>
    <li><strong>Chart.js:</strong> Biblioteca (carregada via CDN) para a criação dos gráficos estatísticos.</li>
    <li><strong>JavaScript (ES6+):</strong> Para toda a interatividade (Guia de 5 Passos, Lightbox da imagem, chamada à API do Gemini).</li>
</ul>
